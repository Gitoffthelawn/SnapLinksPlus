Sites that have given v3 some issues

  • ldraw.org - HTML 4.01 Website, docElem.scrollHeight wasn't valid, switched to using window.scrollX/Y, also docElem.clientHeight wasn't accurate, switching to min(docElem.clientHeight, document.body.clientHeight)
  • nfohump.com - XHTML 1.0 Transitional - min(docElem.clientHeight, document.body.clientHeight) has doc.body.clientHeight as 0 - so testing revealed the current winner is docElem.clientHeight unless > window.innerHeight, then doc.body.clientHeight is accurate.

    Properties that could be of use if further issues occur
    • document.clientHeight
    • docElem.clientHeight
    • doc.body.clientHeight
    • doc.body.parentElement.clientHeight (should be same as docElem.clientHeight)
